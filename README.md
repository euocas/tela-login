# 🎬 Tela de Login - Streaming

Projeto desenvolvido durante a **Imersão Front-End com IA da Alura**, com foco na criação de interfaces modernas utilizando **HTML, CSS e JavaScript**, com apoio de Inteligência Artificial.

## 🚀 Sobre o Projeto

Este projeto consiste em uma **tela de login de plataforma de streaming**, permitindo a seleção entre **4 usuários diferentes**, simulando a experiência de serviços populares.

A proposta foi praticar conceitos fundamentais de desenvolvimento front-end, além de explorar o uso de IA como ferramenta de apoio no desenvolvimento.

## 🧠 Tecnologias e Ferramentas

- **HTML5** → Estrutura da aplicação  
- **CSS3** → Estilização e layout responsivo  
- **JavaScript** → Interatividade e manipulação de elementos  
- **Git & GitHub** → Versionamento de código  
- **VS Code** → Ambiente de desenvolvimento  
- **GitHub Copilot** → Assistência com IA na escrita de código  

## 🤖 Uso de Inteligência Artificial

Durante o desenvolvimento, foi utilizado o **GitHub Copilot integrado ao VS Code**, auxiliando em:

- Sugestões de código
- Autocompletar funções
- Melhoria de produtividade
- Exploração de soluções mais rápidas

## 🎯 Funcionalidades

- Seleção de perfil de usuário
- Interface inspirada em plataformas de streaming
- Layout moderno e organizado
- Interações básicas com JavaScript
